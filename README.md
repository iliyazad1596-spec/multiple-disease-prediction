# 🏥 Multiple Disease Prediction System

## Overview

A machine learning web application that predicts:

- Kidney Disease
- Liver Disease
- Parkinson's Disease

using trained Random Forest models and a Streamlit user interface.

---

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-Learn
- Streamlit
- Pickle

---

## Machine Learning Performance

| Disease | Algorithm | Accuracy |
|----------|------------|------------|
| Kidney Disease | Random Forest | 100.00% |
| Liver Disease | Random Forest | 74.36% |
| Parkinson's Disease | Random Forest | 94.87% |
